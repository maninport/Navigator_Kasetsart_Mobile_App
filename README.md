# Navigator Kasetsart Mobile App(KUTravel) 📍🍃
Navigator Kasetsart is a comprehensive mobile solution designed to enhance the campus experience at Kasetsart University (KU). Whether you are a new student trying to find your faculty, a visitor attending the Kaset Fair, or a staff member looking for the nearest welfare bus, this app serves as your digital companion.

## 🌟 Key Features
* __Interactive Campus Map__: Real-time navigation for buildings, faculties, and landmarks across the KU campus.

* __KU Talai Tracker__: Live tracking of the university's welfare buses (Talai cars) with estimated arrival times.

* __Facility Locator__: Easily find essential services including libraries, computer centers, stadiums, and dormitories.

## 🛠 Tech Stack
* __Frontend__: React Native

* __Backend__: Node.js 

* __Maps API__: Google Maps API

## 🚀 Getting Started
### Prerequisites
* React Native

* Android Studio / Xcode

* An API Key for Google Maps
